OOPS Basic Concepts

There are four basic concepts for Object-Oriented Programming System:

  • Encapsulation
  • Inheritance
  • Abstraction
  • Polymorphism

Knowing each of these concepts can provide an easier understanding on the programming language that utilizes OOPS.

Encapsulation is the concept of wrapping a section of a data or code to make it inaccessible for other codes. They can only be accessed through specific ways that is coded into the program. This helps in having total control in what data is accessed and what is not.

Inheritance is the concept of reusing the code by inheriting the properties and behavior of another program. This is useful for developing programs that utilizes the same operations and lessens developers and programmers time and burden in the coding process.

Abstraction is the concept of hiding relevant complex information and provide only the surface idea. For example, when utilizing a device such as a smartphone, you interact with it through taps and swipes on the screen. You don't need to know the inner workings of how such things work other than how to utilize it for a specific task.

Polymorphism is the concept for one thing to take multiple forms. This typically utilizes the IS-A test. For example, a Girl IS-A Person, IS-A Omnivore, and IS-A Human. Thus, a girl can be considered as a person, an omnivore, or a human.

Variables and Methods

A variable is the storage representation of data. These variables are then divided into different data types that store the kind of information. You can imagine variables as the storage boxes that contains specific information and data types as the labels of the type of information taped on these storage boxes.

int x = 10;
double y = 39.45;
char letter = 'A';
String name = "Java";

The first words in each line are the data types. After those are the names of the variables. The equal operator assigns the value, which are the numbers and letters after it, to the variables. A semicolon is placed after the value to make sure that the instruction of creating this variable is complete. A semicolon is very important to be placed after every line of code to ensure that the compiler knows the instruction is complete. The data types in each line are the different types.

Both int and double are used to assign numerical values. The int data type is used to assign integer values while the double data type is used to assign decimal values. The char and String data types are used to store letter values. The char data type is used to store single letter values while the String data type is used to store word values. The values for these data types must be enclosed with either a single or double quotations. Although it is best habit to enclosed char values with single quotation marks and String values with double quotation marks.

A method, also known as a function, is the behavior or actions of an object. As previously explained, Java is an object-oriented programming language, meaning it utilizes the concepts of objects. If variables store the data of information about the object, then methods are the object's actions and behavior coded in by the programmer. For example, the object Person can run, walk, and talk. The programmer will typically code those actions for the object Person.

void run() {
System.out.println("The person is running!");
}

The small code above is the run() method. This action prints out a message that a person is running.

Classes and Objects

A class is simply the template of an object. It typically contains the properties (variables) and the actions (methods) of the object. A programmer uses this template to create multiple objects of the same type.

An object is the creation or the instance of template (class). A special method called constructor is used to create an object and initialize its variables. A constructor must always have the same name as the class.

class Person {
String name;
int age;
Person (String name, int age) {
this.name = name;
this.age = age;
}
void printPerson() {
System.out.println("Name: " + name);
System.out.println("Age: " + age);
}
}

The code above is class Person to create a Person object. A Person object has two variables name and age. Both name and age are needed to be initialized to create a Person object.

To create an object, the constructor is called to initialize the object's variables.

Person java = new Person("Person", 25);

The first word in the code is the name of the class. The second word is the name of the object. An equal operator is used to assign the values of the object to be stored. The new keyword is used to create objects. Finally, the Person constructor is used to initialize the object's variables. In this case, the values "Person" and 25 will be stored in the name and age variables, respectively. java.printPerson();

For an object to use its method, a dot operator is used. In this case, the java object will use the printPerson() method to print out the name and age of the object.
